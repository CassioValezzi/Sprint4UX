Tela inicial

Mapa interativo para seleção da via. Também é possivel selecionar a via pelo campo "Via Selecionada" ao digitar o nome dela.

Também são disponibilazados dados simples da via como tempo dos semáfaros e trânsito.

Após clicar no botão "Dados da Via"

O usuário será direciondado para a tela que demonstrará dados da via (ou das vias) selecionada tanto para carro como para pedestres (dependendo da seleção), por exemplo: Velocidade média X Tempo da via, Quantidade de carros X Tempo da Via, Trânsito (Quantidade de Carros X m²), Quatidade de pedestres que cruzaram a via X Tempo.

O usuário podera mudar o tempo de temporizador dos semaforos de forma simultanea na via.

Após clicar no botão "Controle Manual"

O usuário será direciondado para a tela que demonstrará os semáforos da via selecinada na "Tela inicial" podendo selecionar quaisquer deles para "Abrir" ou " Fechar", e também mudar o tempo do temporizador de cada semáforo de forma indivudual, além de poder fechar-los para manutenção da via selecionada.
